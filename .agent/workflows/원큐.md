description: (slash_one_queue / 원큐) Automated workflow for Q1 seasonal image generation (Batch Mode of generate-stock-image)

This workflow (`/원큐`) automates the Adobe Stock generation process. It adapts the one-by-one analysis of `@[/generate-stock-image]` into a **batch processing** model.

**Adaptation Note:**
Instead of analyzing each image manually after generation (Step 4 of `/generate-stock-image`), this workflow uses the **pre-defined prompt attributes** (Subject, Theme, Mood) in `seasonal_prompts.json` to generate the required metadata automatically.

### Workflow Steps

#### 0. Environment Check
Ensure the `seasonal_prompts.json` is up to date with your planned prompts.

#### 1. Metadata & Folder Preparation (Crucial Step!)
Creates the session timestamp and the metadata JSON files required for CSV generation later.
**Output:** A timestamp string (e.g., `2026-01-08_12-23-26`). Save this.
```bash
python process_seasonal_metadata.py
```

#### 2. Image Generation (Batch)
**Agent Action:**
Read `seasonal_prompts.json`. For each item:
1.  Run `generate_image` tool.
2.  **Prompt:** Use the `prompt` field (already configured with style/safety rules).
3.  **ImageName:** Use the `filename_base` derived from the ID/Subject (check terminal output of Step 1).

#### 3. Image Moving
Moves the generated images from the agent's artifact folder to the project generation folder.
Replace `<TIMESTAMP>` with the timestamp from Step 1.
```bash
python move_generated_images.py <TIMESTAMP>
```

#### 4. Processing Pipeline (Crop & Upscale)
Automates the **Crop (16:9)** and **Upscale (Real-ESRGAN)** steps.
Replace `<TIMESTAMP>` with the timestamp from Step 1.
```bash
python generation_pipeline.py <TIMESTAMP>
```

#### 5. CSV Generation
Generates `submission.csv` compliant with Adobe Stock metadata guidelines.
Replace `<TIMESTAMP>` with the timestamp from Step 1.
```bash
python generate_submission_csv.py <TIMESTAMP>
```

#### 6. Final Review
Opens the upscaled folder for final visual verification.
Replace `<TIMESTAMP>` with the timestamp from Step 1.
```bash
explorer "generations\<TIMESTAMP>\upscaled"
```
